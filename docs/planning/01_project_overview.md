# 프로젝트 개요

> **프로젝트**: SKN22 Final Project · 2팀
> **버전**: v1.0 (프로토타이핑)
> **작성일**: 2026-03-06

---

## 1. 목적

반려동물(고양이/강아지)을 키우는 사용자가 자연어 대화를 통해 적합한 상품을 추천받을 수 있는 대화형 챗봇 시스템을 구현한다. 사용자의 반려동물 프로필과 대화 맥락을 기반으로 어바웃펫(aboutpet.co.kr) 상품 데이터를 검색·추천한다.

---

## 2. 범위

- 사용자 인증 (회원가입 / 소셜 로그인 / 로그아웃 / 회원 탈퇴)
- 반려동물 프로필 등록 (온보딩)
- 대화형 상품 추천 챗봇 (ChatGPT UX 참고, 3-패널 레이아웃)
- 상품 카드 및 장바구니
- 구매 플로우 (벤더 연동, 프로토타입은 UI까지)
- 사용자 설정 및 프로필 관리
- 게스트 모드
- 어드민 페이지

---

## 3. UX 레퍼런스

ChatGPT 인터페이스를 1차 레퍼런스로 삼는다. 단, 상품 패널을 우측에 추가하는 **3-패널 구조**로 확장한다.

```
┌─────────────┬──────────────────────────┬──────────────────┐
│  사이드바   │       메인 채팅 영역      │   상품 패널      │
│             │                          │                  │
│  [새 대화]  │   AI 메시지 (버블)        │  ┌────────────┐  │
│  ─────────  │                          │  │ 상품 카드  │  │
│  대화 목록  │   사용자 메시지 (우측)    │  │ 이미지     │  │
│             │                          │  │ 타이틀     │  │
│  ─────────  │   ┌────────────────────┐ │  │ 가격/평점  │  │
│  [설정]     │   │ 입력창    [전송 ▶] │ │  │ [+장바구니]│  │
│  [로그아웃] │   └────────────────────┘ │  └────────────┘  │
└─────────────┴──────────────────────────┴──────────────────┘
```
