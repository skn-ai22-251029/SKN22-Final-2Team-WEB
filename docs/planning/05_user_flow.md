# 화면 흐름 (User Flow)

> **프로젝트**: SKN22 Final Project · 2팀
> **작성일**: 2026-03-06

---

## 1. 전체 흐름

```
[랜딩 페이지]
      │
      ├─ 이메일 회원가입 ──► [이메일 인증] ──► [온보딩 STEP 1~3] ──► [완료]
      │                                                                   │
      ├─ OAuth 로그인 (Google / Kakao / Naver) ───────────────────────────┤
      │                                                                   │
      └─ 이메일 로그인 ───────────────────────────────────────────────────┤
                                                                          │
                                                               [메인 챗봇 화면]
                                                                          │
                              [비로그인 접근 시]
                              Input Disabled + "로그인이 필요한 서비스입니다." 표시
                              → 로그인 유도 모달
                                                               [로그인 사용자]
                                                                          │
                                                               [새 대화 / 기존 대화]
                                                                          │
                                                               [추천 질문 칩 표시]
                                                                          │
                                                               [메시지 입력]
                                                                          │
                                                               [AI 응답 + 상품 카드 (우측 패널)]
                                                                          │
                                                               [장바구니 담기 / 추가 질문]
                                                                          │
                                                               [장바구니 → 구매 → 완료]
```

---

## 2. 온보딩 플로우

```
[STEP 1] 반려동물 종류 선택
  강아지 / 고양이 (최대 5마리, append 방식)
  "예비 집사" 옵션 제공 · 스킵 불가
      │
      ▼
[STEP 2] 기본 정보 입력
  이름(필수) · 품종(선택) · 성별(필수) · 나이(필수, 월령 포함)
  체중(선택) · 중성화 여부(선택)
  여러 마리인 경우 탭 전환
      │
      ▼
[STEP 3] 건강 / 식이 입력
  마지막 접종일 · 건강 관심사 · 알레르기 태그
  선호 사료 형태 · 예산대 · 특이사항
      │
      ▼
[온보딩 완료] → 챗봇 메인 페이지 Redirect
  프로필 기반 추천 질문 칩 표시
```

---

## 3. 챗봇 인터페이스 흐름

```
[새 대화 시작]
  사이드바 버튼 클릭 → 대화 대상 펫 프로필 선택
  첫 메시지 전송 시 세션 생성 (Lazy 방식)
      │
      ▼
[메시지 입력]
  텍스트 입력 / 이미지 첨부 (jpeg, png) / 음성 입력 (TBD)
  추천 질문 칩 클릭으로 대화 시작 가능
      │
      ▼
[AI 응답]
  스트리밍 출력 (타이핑 효과)
  응답 생성 중 중단 버튼 지원
  채팅 영역: 추천 근거 텍스트 표시
  우측 패널: 상품 카드 슬라이드 렌더링
      │
      ▼
[상품 카드 인터랙션]
  카드 클릭 → 내부 모달 상세 보기
  (+) 버튼 → 장바구니 담기
      │
      ▼
[장바구니 → 구매]
  결제사 연동 페이지 이동 (프로토타입은 UI만)
  구매 완료 → 챗봇 메인 Redirect · 주문 내역 DB 저장
```
