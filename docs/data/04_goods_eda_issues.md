## 이슈 1 — GO 상품 품절률 완전 0%

### 현상

- `prefix = GO` 상품 621개 전체의 `sold_out_yn = N`
- 다른 접두사 품절률: GI 5.4%, GP 6.3%, GS 9.6%, PI 13.5%

### 원인

GO는 **색상/사이즈 선택형 상품**이다 (예: `라임/화이트`, `S/M`, `2color`).

어바웃펫 시스템에서 GO 상품의 `sold_out_yn`은 **모든 옵션이 동시에 품절일 때만 Y로 전환**되는 구조로 추정된다. 옵션 하나가 품절돼도 다른 옵션 재고가 남아 있으면 N으로 유지된다.

```
예시:
GO251133380 | 크레용펫 세모 미니 칫솔(라임/화이트)
  → 라임 품절 + 화이트 재고 있음 → sold_out_yn = N
```

현재 크롤링 구조는 상품 목록 API(`getScateGoodsList`)의 `data-soldoutyn` 속성을 수집하는데, 이는 **옵션 단위가 아닌 상품 단위** 품절 여부다.

### ETL 대응

- **Silver**: GO 상품에 `soldout_reliable = False` 플래그 추가
- **추천 시스템**: GO 상품의 품절 여부를 신뢰 지표로 사용하지 않음
- **Phase 2**: `indexGoodsDetail` 옵션별 재고 상태 수집으로 개선 가능

## 이슈 2 — GP 상품 리뷰 수 과다 집계 (최대 29.6배)

### 현상

| 접두사 | 식품류 평균 리뷰 수 |
|--------|----------------|
| GI (단품) | 34개 |
| GP (용량묶음) | 996개 |
| **배율** | **29.6x** |

리뷰 수 상위 10개 상품이 모두 GP 또는 PI.

```
GP251077034 | 아카나 캣 와일드 프레이리 모아보기(340g/1.8kg/4.5kg) | 리뷰 24,478개
GP251127447 | 믿고 먹이는 1군 습식 인기템 모아보기                | 리뷰 23,683개
GP251066079 | 이나바 BEST 간식 18종 모음                        | 리뷰 22,854개
```

### 원인

GP는 **"모아보기" 페이지** — 여러 용량/구성의 단품이 하나의 GP ID 하위에 묶인 구조다.

`getGoodsEntireCommentList` API 호출 시 `optGoodsId=""` (빈 값)로 요청하면 **하위 모든 단품의 리뷰가 합산**되어 반환된다. 즉 GP 리뷰 수 = Σ(하위 단품 리뷰 수).

또한 "모아보기"는 단순 용량 묶음이 아닌 **큐레이션 묶음** 형태도 존재한다.
예: `믿고 먹이는 1군 습식 인기템 모아보기` → 수십 개 단품 포함 → 리뷰 23,683개

### ETL 대응

- **Silver**: `review_count_source` 컬럼 추가 — `direct` (GI/GO/GS/PI) vs `aggregated` (GP) 구분
- **Gold**: `popularity_score = log(review_count + 1) × rating` 계산 시 GP는 **goods 목록 API의 `review_count_raw` 그대로 사용** (개별 리뷰 수집 없음 — 아래 참고)
- **추천**: popularity_score 기반 랭킹에서 GP 상품 과대평가 방지

### Bronze 리뷰 수집 제외 결정 (2026-03-10)

**GP 상품은 `bronze_reviews.py` 수집 대상에서 제외한다.**

근거:

| 항목 | 수치 |
|------|------|
| GP canonical 상품 수 | 1,284개 |
| GP `review_count` 합계 | 1,048,748 (하위 GI 리뷰 중복 합산) |
| GI `review_count` 합계 | 48,185 (실제 단품 리뷰) |
| GP 평균 리뷰 수 | 816.8개 |
| GI 평균 리뷰 수 | 21.1개 |
| 1000건+ GP 상품 수 | 132개 |

1. **크롤링 비용**: 1,000건+ GP 132개 × 평균 100페이지 = 수집 시간 폭발
2. **review-product 매핑 모호**: GP 리뷰는 여러 하위 단품의 리뷰가 혼재 → 어느 단품에 대한 감성인지 불분명, Silver에서 `goods_id` 귀속 불가
3. **popularity_score 대체 가능**: `review_count_raw`는 goods 목록 API에서 이미 수집됨

**정보 손실 범위**:

GP의 하위 단품(variant GI)은 우리 카탈로그에 독립 상품으로 존재하지 않는다. 아래 수치가 이를 뒷받침한다:

| | review_count 합계 | 평균 |
|--|--|--|
| GP canonical 1,284개 | 1,048,748 | 816.8 |
| GI canonical 2,277개 | 48,185 | 21.1 |

GP review_count가 GI의 21배 — GP 하위 단품들은 카탈로그 GI와 별개다. 따라서 GP 리뷰를 수집하지 않으면 **GP 전용 리뷰 텍스트는 Qdrant 임베딩에서 완전 누락**된다.

**Phase 1 허용 결정**: GP 상품(1,284개)은 상품명 + 카테고리 태그만으로 임베딩한다. 리뷰 기반 감성 점수 없이 popularity_score는 `review_count_raw × rating` 기반으로 산출한다. Phase 2에서 GP별 개별 variant API 호출(`optGoodsId` 지정)로 단품 리뷰 분리 수집을 검토한다.

> GP 중 "모아보기" 큐레이션 형태(옵션이 아닌 여러 브랜드 묶음)는 보정 불가 → 별도 플래그 필요

## 이슈 3 — 동일 상품명 GI/GP 중복 등록

### 현상

- 동일 상품명 중복: **35개 상품명, 70개 goods_id**
- 가격도 동일 (12개 GI+GP 쌍 모두 가격 일치)

```
중복 조합 패턴:
GP+PI : 14개  (구독 전환 중복)
GI+GP : 12개  (단품+묶음 중복)
GP+GP :  6개  (묶음 중복)
GI+GI :  2개  (단품 중복)
GI+PI :  1개
```

샘플 (GI+GP 쌍):
```
누터스가든 강아지 미니약과 멍약과 수제간식 120g
  [GP] GP251128649 | 가격: 4,250원 | 리뷰: 1
  [GI] GI251128639 | 가격: 4,250원 | 리뷰: 1
```

### 원인

대부분 **누터스가든** 단일 브랜드의 상품 등록 이슈. 동일 상품을 GI(단품)와 GP(묶음 페이지)로 이중 등록한 것으로 추정된다.

`GP+PI` 조합은 기존 GP 상품을 구독(플랜) 상품으로 재등록하면서 발생한 중복이다.

### ETL 대응

- **Silver dedup 정책**: 동일 `(product_name, discount_price)` 조합 중 아래 우선순위로 1개만 유지
  1. GP (리뷰 집계 유리)
  2. PI
  3. GI
  4. GS / GO
- `duplicate_of` 컬럼에 제거된 goods_id 기록 (추적 가능성 확보)
- **추천 시스템**: 중복 상품이 동시에 추천 결과에 노출되지 않도록 필터링

## 이슈 4 — 상세 이미지 50개 초과 상품 (OCR 비용 과다)

### 현상

- 50개 이상 상품: **154개**
- 최대: 162개 (필립스 공기청정기), 130개 (닥터페트라 세트)

```
이미지 수 구간별 분포:
1~10개  : 다수 (정상)
11~20개 : 189건
21~30개 : 다수
31~50개 : 일부
51개+   : 154건 ← OCR 대상 재검토 필요
```

이미지 수 vs 가격 상관계수: **0.294** (약한 양의 상관 — 고가 상품이 이미지 더 많은 경향)

### 원인

특정 브랜드의 **마케팅 전략** 차이가 주요 원인. 가격/카테고리보다 **브랜드별 콘텐츠 정책**에 더 의존한다.

```
50개 이상 브랜드 TOP 10:
조공         19개 상품  (성분/효능 상세 이미지 다수)
닥터페트라    13개 상품  (세트 구성 상세 이미지)
닥터펠리스    11개 상품  (모래 성분/사용법)
개님의상점    10개 상품  (덴탈껌 성분 이미지)
리바펫         9개 상품  (기저귀 사이즈별 이미지)
펫팀           8개 상품  (노즈워크 매트 사용법 102개)
```

펫팀 새싹 노즈워크 매트는 사용법 시연 이미지가 100개 이상으로, OCR 대상 정보(성분/영양소)가 아닌 **마케팅 이미지**가 대부분이다.

### ETL 대응

- **Gold OCR 처리 시 상품당 최대 30개 이미지로 상한 설정**
  - 상한 초과 시 앞 30개만 처리 (상품 상세 페이지 상단이 핵심 정보 위치)
- `ocr_image_count` 컬럼에 실제 OCR 처리한 이미지 수 기록
- Phase 2에서 브랜드별 이미지 패턴 분류 → 마케팅/성분 이미지 자동 구분

## 이슈 5 — OCR 대상 "식품류만" 가정 오류

### 현상

기존 가정: _"상세 이미지(`editor/goods_desc/`)는 식품류 상품에만 존재한다"_

실제 중분류별 상세 이미지 보유율:

| 중분류 | 상품 수 | 이미지 보유율 | 평균 이미지 수 |
|--------|--------|------------|-------------|
| 습식관  | 49     | **93.9%**  | 5.5개 |
| 사료   | 1,405  | **84.6%**  | 7.0개 |
| 모래   | 228    | **80.7%**  | 15.2개 |
| 간식   | 1,321  | **78.5%**  | 7.3개 |
| 덴탈관  | 63     | 55.6%      | 5.6개 |
| 배변용품 | 294    | 51.4%      | 7.0개 |
| **용품** | **1,575** | **36.5%** | 6.1개 |

용품 소분류별 이미지 보유율 (높은 순):

| 소분류 | 보유율 | 비고 |
|--------|--------|------|
| 강아지_용품_건강관리 | **84.3%** | 영양제/보조제 성분 이미지 |
| 고양이_용품_치아관리 | 58.3% | 치약 성분 이미지 |
| 강아지_용품_미용/목욕 | 57.3% | 샴푸 성분 이미지 |
| 강아지_용품_구강관리 | 47.1% | 구강세정제 성분 |

### 원인

기존 가정이 **초기 샘플 조사(10개 미만) 편향**에서 비롯됐다. 전수 수집 결과 용품류 중에서도 영양제, 치약, 샴푸, 구강 관리 제품 등은 **성분표/효능 정보를 이미지로 제공**하는 비율이 높다.

특히 `강아지_용품_건강관리` 84.3%는 사료/간식보다 낮지만 OCR 추출 가치가 높은 카테고리다 — **알레르기 성분, 영양 성분** 정보가 포함된 이미지가 다수 존재하기 때문이다.

### ETL 대응

- **Gold OCR 대상 확장**: 식품류 → 식품류 + 건강관리/치아관리/덴탈관 용품
  ```python
  OCR_TARGET_MIDCATE = ['사료', '간식', '습식관', '덴탈관']
  OCR_TARGET_SUBCATE = [
      '강아지_용품_건강관리',
      '고양이_용품_건강관리',
      '강아지_용품_구강관리',
      '고양이_용품_치아관리',
  ]
  ```
- 위 기준 외 카테고리의 이미지는 `ingredient_text_ocr = NULL` 처리 (OCR 비용 절약)
- `ocr_skipped_reason` 컬럼으로 스킵 사유 기록
